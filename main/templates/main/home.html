Algorithms Used

1

Random Forest

  • Supervised learning algorithm
  • Creates ensemble of decision trees
  • Flexible to classification & regression problems

    • 2

    SVM

  • Supervised learning algorithm
  • Creates the best line /decision boundary so that we can easily put the new data point in the correct category
  • It is memory efficient

    • 3

    Naive Bayes

  • Supervised learning algorithm used for solving classification problems
  • Simple & effective in building fast ML models that can make quick predictions
  • It is highly scalable with the number of predictors and data points
